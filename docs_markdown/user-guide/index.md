# User guide

This guide provides

- material introducing GROMACS
- practical advice for making effective use of
  GROMACS.

For getting, building and installing GROMACS,
see the `/install-guide/index`. For background on algorithms and
implementations, see the
`reference manual part ` of the documentation.
If you have questions not answered by these resources, please visit the
GROMACS [user discussion forum]() and search
for a potential answer or ask a question from the community.
